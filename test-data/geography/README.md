# Fixture geográfica

`ign-ngmep-municipalities-madrid-2026.csv` es una reducción de 29 filas del
fichero `MUNICIPIOS.csv` incluido en `BD_MUNICIPIOS-ENTIDADES.ZIP`, edición
2026-03 del Nomenclátor Geográfico de Municipios y Entidades de Población.

Fuente oficial:
<https://centrodedescargas.cnig.es/CentroDescargas/detalleArchivo?sec=9000004>

- Descarga comprobada: `2026-07-23T15:28:53Z`.
- SHA-256 del ZIP original:
  `496e3079d3b1844e2827d9dfc328fcd2e629e72c2640b46840daa3711b915116`.
- Sistema de referencia: ETRS89, compatible con WGS84 en la península según
  la ficha oficial del producto.
- Licencia: CC-BY 4.0.
- Atribución: Obra derivada de NGMEP 2026 CC-BY 4.0 Instituto Geográfico
  Nacional.

La reducción conserva únicamente código municipal, nombre, longitud, latitud
y origen de coordenada. No contiene información personal ni datos ajenos a los
29 municipios cubiertos por SierraNueva.
